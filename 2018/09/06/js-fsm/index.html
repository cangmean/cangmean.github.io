javascript 有限状态机
2018-09-06|前端js|

有限状态机是一种模型，模拟事物状态的变化，有以下特点

  • 可以用状态来描述事物，并任意时刻事物总是处于一种状态
  • 事物拥有的状态是有限个的
  • 某种条件下，状态可以发生改变，变为另一种状态
